# 🍕 Pizza Sales Analysis (SQL + Power BI)

## 📊 Project Overview

This project analyzes pizza sales data using **SQL for data analysis** and **Power BI for interactive visualization**.
The objective is to extract meaningful business insights such as revenue trends, product performance, peak ordering hours, and customer purchasing behavior.

The project demonstrates practical **data analyst skills including SQL querying, data modeling, and dashboard development**.

## 🧠 SQL Analysis

SQL was used to explore and analyze the dataset to answer key business questions.

### Basic Analysis

* Total number of orders
* Total revenue generated
* Highest priced pizza
* Most common pizza size
* Top 5 most ordered pizzas

### Intermediate Analysis

* Total quantity ordered by pizza category
* Distribution of orders by hour
* Average pizzas ordered per day
* Top pizzas based on revenue

### Advanced Analysis

* Percentage contribution of each pizza type to total revenue
* Cumulative revenue over time using window functions
* Top 3 pizzas by revenue within each category using CTE and ranking

SQL file:

```
pizza_sales_analysis.sql 
```

## 📊 Power BI Dashboard

The Power BI dashboard visualizes the insights derived from SQL analysis.

Dashboard includes:

* KPI cards for revenue and orders
* Monthly order trends
* Peak ordering hours
* Orders by day of the week
* Top 5 best selling pizzas
* Revenue distribution by pizza category
* Orders by pizza size
* Interactive slicers for filtering

---

## 🔍 Key Insights

* Classic pizzas generate the **highest revenue**
* **Large size pizzas** are the most frequently ordered
* Peak order hours occur between **12 PM – 6 PM**
* **Friday and Saturday** have the highest order volume
